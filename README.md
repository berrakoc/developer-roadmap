## 📚 Kategorilerin İçeriği

### 01-Computer-Science
Harvard'ın ünlü CS50 kursu kapsamında, algoritmaların temelleri, veri yapıları ve hesaplamalı düşünme becerilerini geliştiren materyaller.

### 02-Languages
Web ve genel yazılım geliştirme için temel programlama dilleri:
- **Java**: Nesne yönelimli programlama ve kurumsal uygulamalar
- **JavaScript**: Web tarayıcı ve Node.js ile dinamik uygulamalar
- **HTML & CSS**: Web sayfası yapısı ve stilendirme

### 03-Databases
Veritabanları ve SQL öğrenmesi:
- Relational veritabanı tasarımı
- SQL sorgu yazma ve optimizasyon
- Veri yönetim ilkeleri

### 04-Frameworks
Üretim seviyesi web uygulamaları geliştirmek için:
- **Spring Boot**: Java tabanlı mikro servis ve web uygulamaları

### 05-AI-and-Data-Science
Yapay zeka, makine öğrenmesi ve bilgisayarla görme:
- Makine öğrenmesi algoritmaları ve Python uygulamaları
- OpenCV ile görüntü işleme ve görü tabanlı uygulamalar

### 06-Tools-and-Reference
Yazılım geliştirme sürecindeki araçlar ve referans materyalleri:
- **Git**: Versiyon kontrol, branch yönetimi ve işbirliği
- HTML & CSS referans dökümanı
